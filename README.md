**Candidate:** Ankit Kumar
**Assignment:** Product Intern · Vocallabs.ai / Subspace.money

---

## What's inside

A hands-on product teardown of Subspace.money covering 5 feedbacks across 5 product pillars.

---

## Key Findings

- **Homepage is misleading** — opens like a delivery app, hides the core subscription feature
- **No referral loop** — students are the right audience but there's nothing to make them invite friends
- **Admin accountability is broken** — users get removed unfairly with no recourse, hurting ratings
- **Negotiate API is hidden** — a useful bill-negotiation feature exists but is invisible to regular users
- **Local marketplace is empty** — claims to be India's first local subscription marketplace but barely has any local providers

---

## North Star Metric

**Monthly Active Shared-Group Members** — tracks how many users are active in at least one shared subscription group per month

---

## 30-Day Priorities

1. Admin Trust Score + 48hr payment escrow + 24hr refund SLA
2. Rewrite homepage hero copy + fix navigation
3. Launch student referral loop + campus ambassador programme
